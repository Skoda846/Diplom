# Diplom

## Дипломная работа по автоматизации тестирования на python

### Шаги:
1. Склонировать проект 
2. Установить все зависимости
3. Запустить тесты 'pytest'

### Стек:
- pytest
- selenium
- requests
- allure
- config

### Струткура:
- ./test - UI- и API-тесты



### Библиотеки (!)
- pyp install pytest
- pip install selenium
- pip install webdriver-manager

### Как добавить API-ключ
1. Получить API-ключ
Перейти в Telegram и написать боту: @kinopoiskdev_bot
Отправить команду: /start

2. Бот выдаст уникальный API-ключ

3. Скопировать ключ и добавить в код.

### запуск тестов

Запуск всех тестов: pytest

Запуск UI-тестов: pytest -m "ui"

Запуск API-тестов: pytest -m "api"
